# functiongemma-hackathon
Getting started repo for the Cactus x DeepMind Hackathon

## Setup
- Step 1: `git clone https://github.com/cactus-compute/cactus && cd cactus && source ./setup && cd ..`
- Step 2: `cactus build --python`
- Step 3: `cactus download google/functiongemma-270m-it --reconvert`
- Step 4: `python example.py`

Full Python API Reference: [Link](https://github.com/cactus-compute/cactus/blob/main/python/README.md)
